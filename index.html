About

CheeseWhiz SMP is a community-driven survival server built around creativity, progression, and fun. Players can build, explore, trade, and take part in regular events that keep the world active and evolving. With support for both Java and Bedrock, it’s designed to be a place where anyone can join in and help shape the server as it grows.
